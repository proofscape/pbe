Getting Started

Welcome to the Proofscape Browser Extension (PBE)!

Use of this software is under the MIT License. See options page for license text.

The PBE is designed to work with PISE (the Proofscape Integrated Study Environment), allowing PISE to load PDFs from the Internet.

However, first you have to give it permission, in a process we call activation.

Activation

PISE is a web app, which can be hosted by anyone, anywhere on the Internet. For example, there is an instance of PISE running at https://pise.proofscape.org, which supports the Proofscape documentation.

Before the PBE can work with any given instance of PISE, you have to grant it permission to do so, which we call activation. If you have just installed the PBE, it is not yet activated anywhere.

There are several different ways to activate the PBE for a given domain. We will use the instance of PISE running at https://pise.proofscape.org as an example.

The simplest thing is to do nothing right now, and activate later, when the need arises (see Method 0). If you prefer to be more proactive, see Methods 1 or 2.

Activation Method 0: do it later

You actually don't need to do anything right now. The next time you are in PISE, and try to load a PDF which requires the extension's help, you will be shown instructions for what to do. Follow the instructions at that time.

Activation Method 1: from PISE

  1. In a new browser tab, navigate to the site where you want to activate the PBE, such as https://pise.proofscape.org.
  2. While in that tab, click the "Extensions" or "More tools..." button on your browser's toolbar. Depending on your browser, it may have one of the following icons: , , or .
  3. This should bring up a menu where you will see a row for the Proofscape Browser Extension. Click the name of the extension in this row, and follow the prompts in order to activate.

Activation Method 2: from Options Page

  1. When you close this "Welcome" dialog, you will be on the extension's options page. On this page, find the "Activations" section, and click the "+" button to add a new activation.
  2. In the dialog that comes up, enter the URL pattern *://pise.proofscape.org/*, then click "Proceed".

Notes

  • If the PBE does not immediately work with existing PISE tabs after initial activation, try reloading PISE.
  • You only have to activate the PBE once, for a given domain. Thereafter, it will always load when you load PISE from that domain.
